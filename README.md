# WanderStay
WanderStay is a full-stack web application inspired by AirBnb where users can explore property listings, add listings and manage them.

## Features
- View all property listed
- Add new property to list
- Edit the property listed
- Delete property
- Responsive UI
- Server-side rendering using EJS
  
## Tech Stack
HTML
CSS
EJS
Node.js
Express.js
MongoDB
Mongoose
